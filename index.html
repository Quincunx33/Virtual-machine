System Capabilities

High-fidelity emulation powered by libv86 and WASM.

Native Compilation

x86 instructions are dynamically translated to WebAssembly at runtime. This provides near-native execution speed for legacy software and lightweight Linux distributions.

Persistent Storage

Virtual hard drives utilize the browser's IndexedDB API. Create, modify, and save full system snapshots (.v86state) to local storage to resume sessions instantly.

Touch Interface

Designed for mobile devices with a custom virtual keyboard overlay, haptic visual feedback, and an iOS-style assistive touch menu for system interrupts.
