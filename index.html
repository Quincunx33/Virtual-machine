Features

A robust virtual machine platform built with modern web technology.

Flexible Boot Options

Install a new OS from an `.iso` file or boot directly from a pre-configured `.img` disk image. Full control over boot order.

Powerful In-VM Controls

An iOS-style assistive menu gives you access to fullscreen, virtual keyboard, Ctrl+Alt+Del, and other essential functions.

Disk Snapshots

Download a snapshot of your virtual hard disk (`.img` file) with all your changes, ready to be resumed later.

Built-in Disk Explorer

Inspect `.img` files without booting. Our read-only file explorer lets you safely browse FAT16/FAT32 filesystems in-browser.

Secure Sandbox

Runs in a sandboxed browser environment, keeping your local system secure and isolated.

Cloud Storage

Securely connect your Google Drive account to save and load virtual machine disk images directly from the cloud.

About the Project

This project demonstrates the power of modern web technologies by running a full x86 virtual machine directly in the browser. It leverages the libv86 emulation engine and WebAssembly to provide a fast, secure, and accessible computing environment without any server-side components or installations.
